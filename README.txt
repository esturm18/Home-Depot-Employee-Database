Project: Home Depot Employee Database
Purpose: The purpose of this project is a way to keep track of different employees at Home Depot. This program will remove, search, and delete employees. 
To compile: To use this program, please use the makefile. To compile the program simple type 'make' onto a command line and it will compile all the files in the program. To clean up the fiels, simply type 'make clean' and this will remove the executable and the object files. 

When Running: Upon running the program, enter in the store number of any of the available txt files, i.e. 4718 or 4609. This will open the file of that particular store and create a linked list of all the available employees. From there please type in ? into stdin to get access to all the commands you can use in the program.

Thanks for reading!
